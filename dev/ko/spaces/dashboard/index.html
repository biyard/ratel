스페이스 대시보드

대시보드는 호스트의 "내 스페이스, 오늘 어떻게 돌아가고 있나" 화면입니다 — 스탯 카드 그리드와 참여자 랭킹 표가, 라이브 활동에 의해 직접 구동됩니다. 위치는 다음과 같습니다.

/spaces/:space_id/

(끝의 슬래시는 필수입니다 — 대시보드는 catch-all 라우트로 등록되어 있습니다.) 스페이스 탭 에서 열거나 URL 을 직접 붙여넣어 접근합니다. 페이지는 역할에 따라 두 뷰로 분기됩니다. 호스트(Creator) 는 전체 카드를 보고, 참여자와 일반 방문자 는 같은 카드를 읽기 전용으로 보며 자신의 랭킹 행을 찾을 수 있습니다. 같은 URL, 다른 컨트롤 — 분기는 자동입니다.

대시보드가 보여주는 것​

페이지는 화면 크기에 따라 자동으로 적응하는 그리드 (데스크톱은 멀티 컬럼, 모바일은 단일 컬럼) 이며, 데이터가 있을 때 랭킹 표가 그리드 아래에 풀폭으로 노출됩니다.

카드 그리드​

세 개의 카드는 항상 렌더되고, 두 개는 스페이스에 보상/스코어링 데이터가 있을 때만 조건부로 렌더됩니다.

카드항상 렌더?무엇을 보여주는가
요약 (Stat summary)예한눈에 보이는 4 개의 카운터: 총 참여자 · 총 좋아요 · 총 댓글 · 총 액션.
액션 진행 (Action progress)예액션 유형별 카운트 (Poll · Discussion · Quiz · Follow).
참여자 개요 (Participants overview)예헤드라인 참여자 수. 카드 안의 탭형 추이 차트는 오늘 기준 placeholder 입니다 (탭이 비어 있음 — 오늘 시점에 빠진 것 섹션 참고).
사용 가능 포인트 (Total points available)스페이스에 보상 설정이 있을 때만총 포인트 풀 + 행동별 포인트 내역.

갓 만든 스페이스는 항상 렌더되는 3 개 카드가 대부분 0 으로 표시되고, 보상 설정이 있는 스페이스는 포인트 카드가 추가되며, 참여자가 점수를 쌓기 시작하면 그 아래에 랭킹 표가 채워집니다.

랭킹 표​

참여자 한 명이라도 점수를 쌓으면 그리드 아래에 풀폭 랭킹 표가 노출됩니다 (상위 50 명, 페이지당 10 명씩 페이지네이션). 컬럼:

컬럼무엇을 보여주는가
순위 (Rank)리더보드에서의 위치, 점수 쿼리에서 라이브로 재계산. 클릭으로 정렬하는 동작은 (예정).
참여자 (Participant)아바타 + 디스플레이 네임.
점수 (Score)이 스페이스에서 누적한 점수. 클릭으로 정렬하는 동작은 (예정).

표 하단의 페이지 단위 페이저로 추가 페이지를 넘길 수 있습니다 — 리더보드가 깊은 스페이스에 유용합니다.

팁. 호스트는 리포트 를 만들기 전에 랭킹 표로 핵심 기여자를 먼저 파악합니다. 참여자 점수는 Phase 4 매출 분배가 출시되면 (예정) 기여자 몫의 가중치로 활용됩니다.

대시보드 / 오버뷰 / 리포트, 언제 무엇을?​

스페이스에는 호스트가 쓰는 화면이 몇 가지 있고, 라이브 통계가 있는 화면은 대시보드뿐입니다. 언제 무엇을 열어야 하는지 빠르게 정리합니다.

화면URL / 트리거언제 열까
대시보드/spaces/:space_id/현재 스페이스가 어떻게 돌아가고 있는지 실시간 수치로 확인 할 때 사용합니다. 지금까지 몇 명이 참여했는지, 어떤 액션이 얼마나 완료됐는지, 점수 랭킹은 어떻게 되는지가 한눈에 들어오므로, 스페이스가 활발히 운영되는 동안에는 하루에 여러 번 열어 흐름을 점검하기 좋습니다.
오버뷰 패널상단바 file-text 아이콘 (URL 없음 — 아레나 위로 슬라이딩)내러티브를 편집 할 때 — 이 스페이스가 무엇이고, 누구를 위한 곳이고, 왜 참여해야 하는지. Creator 는 인라인 편집기를 보고, 뷰어는 read-only 로 봅니다.
리포트/spaces/:space_id/report/활동을 종합한 장문의 AI 보조 리포트를 발행 할 때. 리포트 챕터 참고.
아레나 뷰어/spaces/:space_id/첫 방문자가 도착하는 스플래시 화면을 확인 할 때 — 참여 흐름과 참여 후 액션 카루셀까지. QA 점검에 유용합니다.

대시보드는 활동이 들어오면 자동으로 갱신되는 유일한 화면이고, 나머지는 호스트가 사람들에게 무엇을 말하는가 와 관련된 화면입니다.

오늘 시점에 빠진 것​

대시보드에 아직 없는 항목들 (각각 로드맵에 있음):

  • 액션 단위 드릴다운 (예정). 참여 액션 카드의 진행 바는 읽기 전용 요약입니다 — 바를 눌러 "이 폴에 응답한 12 명은 누구인가" 같은 드릴다운은 아직 없습니다. 그 용도로는 Analyzes 앱 을 사용하시기 바랍니다.
  • 기간 선택기 (예정). 현재 숫자는 누적값입니다 — "최근 7 일" / "이번 사이클" 필터는 로드맵에 있습니다.
  • 익스포트 (예정). 대시보드 스냅샷의 CSV / PDF 익스포트. Analyzes 앱에는 분석별 엑셀 익스포트가 이미 있습니다.

다음 단계​

  • 스페이스 앱 — 대시보드의 보상 카드를 채우는 인센티브 풀을 설치하시기 바랍니다.
  • 호스트 액션 — 진행 바를 채우는 폴 · 토론 · 퀴즈 · 팔로우를 만드시기 바랍니다.
  • 리포트 — 대시보드 활동을 발행 가능한 내러티브로 바꾸시기 바랍니다.
