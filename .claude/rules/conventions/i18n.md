# i18n (Internationalization)

## Rule: All user-facing strings must use `translate!` macro

```rust
translate! {
    MyTranslate;
    title: { en: "Title", ko: "제목" },
}
// In component:
let t = MyTranslate::new(use_locale());
rsx! { "{t.title}" }
```

Define translations in `i18n.rs` within the component/feature directory.

## Rule: Enum values in UI — `.translate()`, never `.to_string()`

```rust
#[derive(Translate)]
pub enum Status {
    #[translate(en = "Active", ko = "활성")]
    Active,
}
// Use: {status.translate(&lang())}
```
