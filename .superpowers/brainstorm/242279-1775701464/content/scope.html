Inside the Space — Action UI proper
1

Action list page (Quest Map)

Replace the grid of cards with a vertical quest map. Per-space progress bar at the top, locked/unlocked states, action cards become quest tiles. The headline change.

2

Quest briefing screen

New "briefing" overlay before participation: shows stakes, reward (+XP, +P, combo bonus), time window, prerequisites, retry policy. Replaces today's quiz "Overview" tab + adds it to polls/discussion/follow.

3

Action player (poll / quiz / discussion / follow)

Restyle the question-by-question interaction with arcade-bold treatment: progress dots, combo meter, animated answer feedback, big chunky bottom bar.

4

Completion / level-up moment

The dopamine peak: animated +XP popup, combo break/extend, streak bump, "you unlocked Quest 4" reveal, optional level-up overlay if you crossed a threshold.

5

Per-space leaderboard (full page)

Today's small ranking widget gets a full-page treatment: podium for top 3, scrollable rest, "you are here" highlight. Lives at /spaces/{id}/leaderboard or as a tab on the action page.

Cross-space — Global RPG layer
6

Global player profile (level, XP, class)

A persistent identity card that shows up in the navbar / user menu and on a profile page. Aggregates XP from every space. (Backend: new global score table.)

7

Achievements collection

Unlockable badges across all spaces — "Voted in 10 polls", "First perfect quiz", "Completed all actions in a Space". A new page that lists owned + locked achievements.

8

Seasons / leagues

Bronze → Silver → Gold → Diamond tiers, weekly/monthly resets. Separate seasonal XP that decays. New backend.

9

Skill tree (Voter / Debater / Scholar / Networker)

Each action type levels its own track. Browseable as a tree. New backend (per-category XP).

10

Global cross-space leaderboard

Top players across the entire platform. Filter by season / category. Lives at /leaderboard.

Click every box you want in V1. My recommendation: 1-5 in V1 (Space-scoped), with 6 (player profile) as a stub using existing per-space score → so the rest can ship later as a clean expansion. But you decide.
