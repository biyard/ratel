How deep should the gamification go?

Pick the depth that matches the energy you want users to feel. We can dial up or down later — but this sets the budget for new mechanics.

A

Light Polish — "Make it feel alive"

Same mechanics, better feedback. Reuses every existing primitive (points, score, ranking). Adds animation, celebration, progress, and visual hierarchy on top.

  • Animated +XP popup when you submit a poll/quiz
  • Progress bar "3 of 7 actions completed in this Space"
  • Action cards get difficulty hints, hover micro-interactions, finished-state checkmarks
  • Confetti / level-up moment after the last action of a Space
  • Ranking widget gets a real podium and "you moved up" indicator

Cost: ~1-2 weeks · No backend changes · Reversible

B

Quest Journey — "Turn actions into a story"

Reframe actions as a guided journey. Cards become quests in an ordered storyline; locked/unlocked states; chapter progress; persistent meta-progression bar at the top.

  • Action list becomes a quest map / vertical timeline instead of a grid
  • Sequential or branch unlocks — finishing Quest 2 reveals Quest 3
  • Chapter rewards — completing a chapter mints a badge / NFT-style proof
  • Rich "quest briefing" screen before starting (stakes, time, reward visible upfront)
  • Streak tracker — daily / consecutive participation

Cost: ~3-4 weeks · Light backend (chapters, unlocks) · Bigger UX rewrite

C

RPG Arena — "Full-on game layer"

Persistent identity across spaces. A user has a level, an XP curve, a skill tree, achievements, seasonal leagues. Spaces feel like dungeons inside a larger meta-game.

  • Player profile card with level, XP bar, class/role
  • Achievements & badges across all spaces (collection page)
  • Seasons / leagues — Bronze → Gold → Diamond, weekly resets
  • Skill tree per action category (Voter, Debater, Scholar...)
  • Combo / multiplier mechanics — 3 actions in a row = 1.5× points
  • Leaderboard becomes a full page, not a side widget

Cost: ~6-10 weeks · Significant backend (cross-space identity, seasons, achievements) · Coordinated launch
