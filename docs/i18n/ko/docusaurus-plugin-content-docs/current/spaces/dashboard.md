# 스페이스 대시보드

대시보드는 호스트의 _"내 스페이스, 오늘 어떻게 돌아가고 있나"_ 화면입니다 — 스탯 카드 그리드와 참여자 랭킹 표가, 라이브 활동에 의해 직접 구동됩니다. 위치는 다음과 같습니다.

```
/spaces/:space_id/
```

(끝의 슬래시는 필수입니다 — 대시보드는 catch-all 라우트로 등록되어 있습니다.) [스페이스 탭](./#스페이스-안의-탭) 에서 열거나 URL 을 직접 붙여넣어 접근합니다. 페이지는 역할에 따라 두 뷰로 분기됩니다. **호스트(Creator)** 는 전체 카드를 보고, **참여자와 일반 방문자** 는 같은 카드를 읽기 전용으로 보며 자신의 랭킹 행을 찾을 수 있습니다. 같은 URL, 다른 컨트롤 — 분기는 자동입니다.

## 대시보드가 보여주는 것

페이지는 화면 크기에 따라 자동으로 적응하는 그리드 (데스크톱은 멀티 컬럼, 모바일은 단일 컬럼) 이며, 데이터가 있을 때 **랭킹** 표가 그리드 아래에 풀폭으로 노출됩니다.

갓 만든 스페이스는 항상 렌더되는 3 개 카드가 대부분 0 으로 표시되고, 보상 설정이 있는 스페이스는 포인트 카드가 추가되며, 참여자가 점수를 쌓기 시작하면 그 아래에 랭킹 표가 채워집니다.

### 랭킹 표

참여자 한 명이라도 점수를 쌓으면 그리드 아래에 풀폭 **랭킹** 표가 노출됩니다 (상위 50 명, 페이지당 10 명씩 페이지네이션). 컬럼:

| 컬럼                     | 무엇을 보여주는가                                                                      |
| ------------------------ | -------------------------------------------------------------------------------------- |
| **순위 (Rank)**          | 리더보드에서의 위치, 점수 쿼리에서 라이브로 재계산. 클릭으로 정렬하는 동작은 _(예정)_. |
| **참여자 (Participant)** | 아바타 + 디스플레이 네임.                                                              |
| **점수 (Score)**         | 이 스페이스에서 누적한 점수. 클릭으로 정렬하는 동작은 _(예정)_.                        |

표 하단의 페이지 단위 페이저로 추가 페이지를 넘길 수 있습니다 — 리더보드가 깊은 스페이스에 유용합니다.

> **팁.** 호스트는 [리포트](./reports) 를 만들기 전에 랭킹 표로 핵심 기여자를 먼저 파악합니다. 참여자 점수는 Phase 4 매출 분배가 출시되면 _(예정)_ 기여자 몫의 가중치로 활용됩니다.

## 대시보드 / 오버뷰 / 리포트, 언제 무엇을?

스페이스에는 호스트가 쓰는 화면이 몇 가지 있고, 라이브 통계가 있는 화면은 대시보드뿐입니다. 언제 무엇을 열어야 하는지 빠르게 정리합니다.

| 화면                                                                                                                                                                                                                                                                                                              | URL / 트리거                                              | 언제 열까                                                                                                                                                |
| ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------- |
| <img src={useBaseUrl('/img/icons/grid.svg')} width="14" alt="대시보드" style={{verticalAlign: 'middle'}} /> **대시보드**                                                                                                                                                                                          | `/spaces/:space_id/`                                      | **현재 스페이스가 어떻게 돌아가고 있는지 실시간 수치로 확인** 할 때 사용합니다. 지금까지 몇 명이 참여했는지, 어떤 액션이 얼마나 완료됐는지, 점수 랭킹은 어떻게 되는지가 한눈에 들어오므로, 스페이스가 활발히 운영되는 동안에는 하루에 여러 번 열어 흐름을 점검하기 좋습니다. |

대시보드는 활동이 들어오면 자동으로 갱신되는 유일한 화면이고, 나머지는 _호스트가 사람들에게 무엇을 말하는가_ 와 관련된 화면입니다.

## 오늘 시점에 빠진 것

대시보드에 아직 없는 항목들 (각각 로드맵에 있음):

- **액션 단위 드릴다운** _(예정)_. 참여 액션 카드의 진행 바는 읽기 전용 요약입니다 — 바를 눌러 _"이 폴에 응답한 12 명은 누구인가"_ 같은 드릴다운은 아직 없습니다. 그 용도로는 [Analyzes 앱](./apps#-analyzes) 을 사용하시기 바랍니다.
- **기간 선택기** _(예정)_. 현재 숫자는 누적값입니다 — _"최근 7 일"_ / _"이번 사이클"_ 필터는 로드맵에 있습니다.
- **익스포트** _(예정)_. 대시보드 스냅샷의 CSV / PDF 익스포트. Analyzes 앱에는 분석별 엑셀 익스포트가 이미 있습니다.
