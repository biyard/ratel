# 팀

Ratel 의 **Team** 은 자체 핸들, 자체 아레나 페이지, 자체 트레저리를 가진 다인 협업 단위입니다. 함께 만든 결과물은 개인이 아닌 **팀의 이름**으로 남고, 팀은 게시글을 발행하고, 보상이 걸린 스페이스를 운영하고, 수익을 분배하고, 내장된 DAO 로 스스로를 통치합니다.

가까운 미래에는 모든 팀이 통합된 **Essence House** — 팀이 무엇을 지향하고 어떤 지식을 쌓아왔는지 보여주는 공유 공간 — 를 가지게 됩니다. 지금 단계에서는 팀을 "거버넌스가 기본 내장된 협업 단위" 라고 생각하시면 됩니다.

이 챕터는 독자에 따라 세 페이지로 나뉘어 있어요.
