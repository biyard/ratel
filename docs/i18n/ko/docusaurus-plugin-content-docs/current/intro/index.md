# Ratel 소개

**Ratel은 Human Essence Platform입니다.** 실제 사람의 생각, 통찰, 그리고 활동을 기록하고 임베딩하여, *직접 활용*하고 *공유*하며 *수익화*할 수 있는 개인 자산으로 만들어 주는 플랫폼입니다. 여러분이 작성한 게시글, 참여한 Space, 던진 투표, 남긴 댓글과 토론은 모두 **여러분의 Essence(에센스)** — 참여할수록 자연스럽게 자라나는 개인 지식 베이스 — 의 일부가 됩니다.

Essence는 온전히 여러분의 자산입니다. 좋아하는 AI 어시스턴트에서 바로 질의하고, ChatGPT나 Claude Desktop에 연결하고, 에이전트를 통해 대신 활동하게 하거나, 자신만의 **Essence House**를 통해 구독 형태로 판매할 수 있습니다.

## Ratel에서 할 수 있는 일

- **🧠 실제 활동에서 Essence 만들기.** 게시글, 댓글, 투표, Space 활동이 자동으로 임베딩되어 개인 지식 베이스로 쌓입니다. AI가 작성한 콘텐츠는 기본적으로 제외되며, 원하는 소스만 직접 켤 수 있습니다.
- **🔌 평소 쓰던 도구와 연결.** **Essence Connector**를 통해 이미 사용 중인 플랫폼(Notion 우선 지원, 이후 Google Docs · Substack · X · Slack 추가 예정)에서 read-only OAuth와 실시간 웹훅으로 콘텐츠를 끌어옵니다. 기존 지식을 다시 옮겨 적을 필요가 없습니다.
- **🏠 Essence House 열기.** *(예정)* 모든 사용자에게 자신의 Essence를 묶어주는 House가 자동으로 생성됩니다. 구독 가격은 직접 정합니다. 구독자는 ChatGPT, Claude Desktop 등 MCP 호환 클라이언트에 등록할 **단일 통합 MCP 엔드포인트**를 받게 되며, 이 URL 하나로 구독 중인 모든 House에 동시 접근할 수 있습니다.
- **🤖 에이전트로 패시브 인컴.** *(예정)* 자신의 Essence로 에이전트를 만들어, 보상이 걸린 Space — 폴, 토론, 퀴즈, 팔로우 퀘스트 등 — 에 참여시킬 수 있습니다. 호스트는 에이전트 참여를 허용하거나 차단할 수 있고, 품질 점수와 직접 활동 지표가 신호의 질을 유지합니다.
- **📊 리포트 수익 공유.** *(예정)* Space 호스트가 리포트를 발행하면 판매 수익은 **플랫폼 10% · 호스트 60% · 기여자 30%** 로 분배됩니다. 기여자 몫은 각자의 활동이 최종 리포트에 얼마나 반영됐는지에 따라 가중치가 적용됩니다.
- **🔒 내 Essence, 내 규칙.** Essence는 사용자의 자산이며 언제든 삭제할 수 있습니다. 구독 판매는 *추론(inference) 접근권*만을 대상으로 하며, 원본 게시글이나 원본 콘텐츠 자체는 절대 판매되지 않습니다.

Ratel은 분산형 입법 플랫폼에서 출발해, **사람의 생각이 곧 결과물이 되는 모든 집단**을 위한 플랫폼으로 진화했습니다. 기반이 되는 핵심 요소 — Space, 게시글, 폴, 토론, 보상, MCP — 는 그대로지만, 이제는 한 사람의 Essence를 일급(first-class)이며 이식 가능하고 수익화 가능한 자산으로 만든다는 더 큰 목표를 향해 움직입니다.

## 다음으로 읽을 곳

이 가이드는 첫 로그인부터 자신의 커뮤니티를 운영하는 단계까지 차례대로 안내합니다.

| 챕터 | 내용 |
|---|---|
| 2. 시작하기 | 회원 가입, 프로필 설정, DID 연결, 첫 게시글 작성. |
| 4. 팀 | Team 또는 하위팀(Sub-team)을 만들고, 멤버를 초대하고, 게시 권한과 Space를 함께 운영하기. **하위팀** (상위 / 하위 라이프사이클) 과 **팀 설정** (Team 계정 관리자 투어) 도 이 챕터 안에 포함됩니다. |
| 5. 게시글 | 글쓰기, 댓글, 투표, 크로스 포스팅 — 그리고 각 활동이 Essence에 어떻게 반영되는지. |
| 6. Space | Space를 호스팅하거나 참여하고, 토론·폴·퀴즈·팔로우 퀘스트를 운영하고, 보상을 설정하기. |
| 7. 에센스 | 어떤 활동이 Essence 에 쌓였는지 확인하고, 소스를 관리하고, MCP 로 AI 에 연결합니다. |
| 8. 보상 | 다섯 가지 보상 단위 — XP, 포인트, 크레딧, 토큰, 인센티브 풀 — 와 그 흐름. |
| 9. 자격증명 | 본인의 속성을 인증 — KYC + 오프라인 코드 — 하고, 인증한 것으로 스페이스를 게이팅. |
| 10. 아케이드 | 3분 분량의 판단 + 글쓰기를 칩 보상과 Essence 엔트리로 바꿔주는 실시간 스킬 게임들. **Fact or Fold** 부터 시작. |

## 현재 제공되는 기능

Ratel은 단계적으로 기능을 출시합니다. 게시글, 댓글, 투표, Space, Team, 본인 Essence 대상 MCP는 오늘 바로 사용할 수 있고, **Notion Essence Connector**는 Phase 1 단계 — *(예정)* — 입니다. 마켓플레이스 기능(**Essence House** 구독 마켓플레이스, Space용 **에이전트**, **리포트 수익 공유**)은 로드맵에 있으며, 본 문서 전반에서 명확히 **"(예정)"** 으로 표시됩니다. 이 라벨이 보이면 기반은 이미 갖춰져 있고, 사용자용 화면이 곧 도착한다는 의미입니다.

Ratel에 오신 것을 환영합니다. 여러분의 Essence를 함께 만들어 봅시다.
