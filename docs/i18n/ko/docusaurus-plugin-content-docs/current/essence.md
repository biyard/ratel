# Essence 소스

Essence 에는 컨트롤 패널이 있습니다 — Essence 에 들어가는 모든 입력을 한 페이지에서 보고, 검색하고, 더 이상 원하지 않는 항목을 정리할 수 있어요. 주소는 다음과 같습니다.

```
/essence
```

[첫날 체크리스트](./getting-started.md#%EC%B2%AB%EB%82%A0-%EC%B2%B4%ED%81%AC%EB%A6%AC%EC%8A%A4%ED%8A%B8), 인앱 내비게이션, 또는 URL 직접 입력으로 언제든 열 수 있습니다.

## `/essence` 가 보여주는 것

페이지는 위에서 아래로 네 영역으로 나뉩니다.

1. **히어로** — 라이브러리 전체의 소스 수와 단어 수.
2. **분포(Breakdown)** — 같은 합계를 소스 종류별로 분해 (필터로도 동작).
3. **컨트롤** — 검색과 정렬.
4. **소스 테이블** — 소스 한 건당 한 행, 행 단위 / 일괄 작업 포함.

Ratel 에 본인이 작성하거나 호스트한 모든 것 — 게시물, 게시물 댓글, 토론 댓글, 본인이 호스트한 폴 / 퀴즈 — 이 자동으로 색인됩니다. 직접 추가하지 않아도 — 활동하면 알아서 등장합니다.

## 히어로 — 한눈에 보는 라이브러리

히어로는 한 문장처럼 읽힙니다.

> **`N` 개 소스 · `M` 단어** *(예시 — 실제 숫자는 본인 라이브러리에 따라)*

이 숫자는 *내 Essence 전체* 의 공식 합계입니다 — 테이블에 지금 보이는 행만의 합계가 아니에요. 단어 수는 모든 소스의 실제 본문을 합산해서 나옵니다.

## 분포 — 소스 종류별 분배

히어로 아래에는 종류별 카드 행이 있습니다. 각 카드는 개수와 전체 대비 비율(%) 을 보여주고, 동시에 필터 칩 역할을 합니다 — 카드를 누르면 아래 테이블이 그 종류로 좁혀집니다.

- <img src={useBaseUrl('/img/icons/grid.svg')} width="16" height="16" alt="전체" style={{verticalAlign: 'middle'}} /> **전체(All)** — 필터 없이 모든 소스.
- <img src={useBaseUrl('/img/icons/notion.svg')} width="16" alt="Notion" style={{verticalAlign: 'middle'}} /> **Notion** *(예정)* — 연결된 Notion 워크스페이스의 페이지.
- **포스트(Posts)** — 발행한 모든 게시물.
- **댓글(Comments)** — 게시물 아래의 댓글과 토론 스레드 안의 댓글 (여기서는 함께 카운트).
- **투표(Polls)** — 본인이 호스트한 폴 (응답자 수와 무관하게, 호스트의 Essence 에 한 번만 색인됩니다).
- **퀴즈(Quizzes)** — 본인이 호스트한 퀴즈.

카드를 누르면 필터가 적용되고, **전체** 를 누르면 해제됩니다.

## 컨트롤 — 검색과 정렬

테이블 위에는 다음이 있어요.

- **검색창** — 제목, 본문, 태그로 매칭. macOS 에서 `⌘K`, 그 외 환경에서 `Ctrl K` 단축키로 포커스가 이동합니다.
- **정렬** — **최근 편집 ↓**, **단어 수 ↓**, **제목순(A–Z)** 중 선택.

검색은 입력하는 동안 실시간으로 결과가 업데이트됩니다. 정렬은 현재 필터·검색 결과에 그대로 적용됩니다.

## 소스 테이블 — 행 단위 컨트롤

각 행은 다음을 보여줍니다.

- 소스 **제목**.
- **종류 뱃지** — *포스트*, *투표*, *퀴즈*, *포스트 댓글*, *토론 댓글*, *Notion*. 댓글에는 부모 컨텍스트 태그가 함께 붙어, 게시물 스레드의 댓글인지 토론 스레드의 댓글인지 구분이 됩니다.
- **단어 수**.
- **최근 동기화 시각** — 이 소스가 Essence 에 마지막으로 갱신된 시점.
- 행 단위 작업이 들어 있는 **`…` 메뉴**.

행 메뉴는 현재 **삭제(Delete)** 를 제공합니다 — Essence 색인에서 그 소스를 제거합니다. 원본 자체(게시물, 댓글, 폴) 는 플랫폼에 그대로 남아 있고, Essence 색인에서만 빠집니다.

테이블 아래 페이지네이션은 "Showing X – Y of Z" 와 **이전 / 다음** 버튼을 보여줍니다. 컬럼이나 칩을 누르면 새로운 뷰의 첫 페이지로 리셋됩니다.

## 일괄 작업 *(예정)*

행 단위 체크박스 + 하단 플로팅 바 — **`N` 개 선택됨 · 제거(Remove)** 로 동작하는 일괄 선택 워크플로우는 로드맵에 있습니다. 행을 하나라도 선택하면 바가 나타나고, **제거** 가 선택된 모든 소스를 Essence 색인에서 한 번에 삭제합니다.

이 기능이 출시되기 전까지는 행 단위 **`…` → 삭제** 메뉴로 한 건씩 제거하세요.

**오늘 사용 가능** (Ratel 네이티브 소스)

| 종류 | 무엇을 담는가 |
|---|---|
| 포스트 | 발행한 모든 게시물. |
| 포스트 댓글 | 게시물 아래에 남긴 댓글. |
| 토론 댓글 | 스페이스의 토론(Discussion) 액션에 남긴 댓글. |
| 투표 | 본인이 호스트한 폴 (호스트의 Essence 에 한 번 색인). |
| 퀴즈 | 본인이 호스트한 퀴즈. |

**Phase 1** *(예정)*

| 종류 | 무엇을 담는가 |
|---|---|
| Notion | 연결된 Notion 워크스페이스의 페이지, 웹훅으로 실시간 동기화. |

**그 이후** *(로드맵, 아직 미착수)*

README 의 커넥터 로드맵에 따라: Google Docs, Substack, X, Slack. 아직 어떤 형태로도 구현되지 않은 항목들로, Notion 출시 후 Essence Connector 프레임워크가 일반화된 시점 이후에 단계적으로 도입될 예정입니다.

Ratel 자체 소스는 활동하면 자동으로 채워집니다. 외부 소스는 **Essence Connector** 를 통해 들어옵니다 — 읽기 전용 OAuth 와 실시간 웹훅을 사용하며, 동기화가 시작되기 전에 설정에서 한 번 연결해 두면 됩니다.

## 프라이버시와 통제

Essence 는 내 자산입니다. 페이지가 보장하는 원칙은 다음과 같아요.

- **삭제 가능.** 모든 소스에는 행 메뉴의 삭제가 있습니다 — 제거된 소스는 색인에서 빠집니다. (일괄 제거(Bulk Remove) 는 *(예정)* 입니다 — 위 섹션 참고.)
- **AI 작성 콘텐츠 옵트인** *(예정)*. AI 도구로 작성한 콘텐츠를 Essence 에서 기본적으로 제외하고, 소스별 옵트인을 통해 AI 보조 글이 Essence 에 들어오도록 허용하는 정책이 의도된 모습입니다. 이 플래그는 오늘 시점에는 아직 동작하지 않으며, 정책 자체는 Phase 1 에 함께 출시됩니다.
- **추론(inference) 접근만.** Essence House 마켓플레이스가 출시되면 *(예정)*, 구독자는 추론 접근권을 사는 것이지 원본을 다운로드하는 게 아닙니다. `/essence` 에서 소스를 제거하면 추론에서도 빠집니다.

내 House 가 실제로 나에 대해 무엇을 알고 있는지 확인하고 싶을 때 — `/essence` 를 열어, 답이 마음에 드는 모양이 될 때까지 다듬으세요.
