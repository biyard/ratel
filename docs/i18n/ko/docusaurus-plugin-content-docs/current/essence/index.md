# 에센스

본인의 **에센스(Essence)** 는 Ratel 에서 한 활동에서 자라나는 개인 지식 베이스입니다 — 발행한 게시글, 작성한 댓글, 호스팅한 폴이나 퀴즈, 스페이스에서 완료한 모든 액션. 플랫폼은 이 각각을 **EssenceSource** 로 변환합니다 — 검색을 위해 인덱싱되고 본인 계정에 귀속되며 (이후 단계에서는) AI 어시스턴트가 질의할 수 있도록 벡터 표현으로 임베딩되는 구조화된 기록이에요.

두 가지 점수 신호가 본인의 Essence 를 플랫폼이 — 그리고 이후에는 마켓플레이스가 — 어떻게 읽을지를 결정합니다.

- **품질 점수(Quality Score)** — 기여의 *질* 을 측정합니다 (깊이, 독창성, 동료 반응). 잘 다듬어진 긴 토론 답변이 폴이나 팔로우보다 높게 평가됩니다.
- **직접 활동 지표(Direct-Activity Index)** — 본인의 시간이 수동 탐색이 아니라 직접적 · 의도적 기여로 들어간 정도를 측정합니다. 글쓰기, 투표, 토론이 스크롤보다 더 무겁게 잡혀요.

두 신호 모두 본인의 Essence 가 검색에서 어디에 노출되는지에 영향을 주고, (Phase 2 에서는) Essence House 구독자가 질의할 때 받는 답에도 영향을 줍니다.

미리 알아두면 좋은 원칙들:

- **에센스는 본인의 자산입니다.** 모든 소스에는 Delete 컨트롤이 있고, Essence 인덱스에서 소스를 제거하면 추론(inference) 에서도 빠집니다. 원본 (게시글, 댓글) 은 플랫폼에 그대로 남고, Essence 인덱싱만 빠져요.
- **추론 접근권만 판매됩니다.** Essence House 마켓플레이스가 출시되면 *(예정)* 구독자는 *추론 접근권* 을 삽니다 — Essence 에 질의할 수는 있지만 원본 콘텐츠를 다운로드할 수는 없어요.
- **AI 작성 콘텐츠는 옵트인입니다.** *(예정)* AI 가 생성한 텍스트는 기본적으로 Essence 에서 제외되며, AI 보조 글쓰기를 Essence 에 반영하려면 소스별로 직접 켜야 합니다.
