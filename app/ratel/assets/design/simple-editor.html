Simple Editor
contenteditable + minimal toolbar — replaces Tiptap

Welcome to Simple Editor

This is a self-built rich text editor that replaces Tiptap. It uses native contenteditable + a small JS shim, so it's tiny, IME-safe, and works inside Android/iOS WebViews.

Try the toolbar above — bold, headings, lists, alignment, links, images, YouTube embeds, tables, and code blocks all work.

Tip: paste rich content from any source — formatting is preserved.
// Inline code and code blocks are supported

  return "world";
