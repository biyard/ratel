Claude Code is Anthropic's CLI agent. Add Ratel as a streamable HTTP MCP server in one command, then call /mcp inside Claude Code to confirm.

1
Add the server with claude mcp add
Run this in your terminal. Replace the URL with the one you copied above.
terminal 
claude mcp add --transport http ratel \
  https://ratel.foundation/mcp/r_l8fk2p9q4nXJv2K3mQpYvHbWcZxRtAuB1eFgHiJkLm
2
Verify the connection
Inside an active Claude Code session, run /mcp. You should see a ratel server with status ✓ connected and the available tools listed.
3
Try it out
Ask Claude Code something like:
“List my latest 5 Ratel posts and mark all unread notifications as read.”
Local-only token
Claude Code stores the URL in ~/.claude.json on your machine. Don't commit this file to git.

Claude Desktop reads its MCP server list from claude_desktop_config.json. Add the snippet below, save, and fully quit + relaunch Claude.

1
Open the config file
macOS: ~/Library/Application Support/Claude/claude_desktop_config.json
Windows: %APPDATA%\Claude\claude_desktop_config.json
2
Add the ratel server
Merge into "mcpServers" (create the key if missing).
claude_desktop_config.json 
{
  "mcpServers": {
    "ratel": {
      "transport": "http",
      "url": "https://ratel.foundation/mcp/r_l8fk2p9q4nXJv2K3mQpYvHbWcZxRtAuB1eFgHiJkLm"
    }
  }
}
3
Restart Claude Desktop
Fully quit (not just close) then reopen. Look for the 🔌 icon at the bottom of the chat input — ratel should appear in the connected servers list.

Cursor supports MCP servers via Settings → Cursor Settings → MCP. You can either add the server through the UI or paste the JSON directly.

1
Open Cursor Settings · MCP
Press ⌘, (macOS) or Ctrl+, (Windows/Linux), then search "MCP".
2
Add a new server  →  HTTP / Streamable
Name: ratel · URL: paste the endpoint you copied above.
~/.cursor/mcp.json 
{
  "mcpServers": {
    "ratel": {
      "url": "https://ratel.foundation/mcp/r_l8fk2p9q4nXJv2K3mQpYvHbWcZxRtAuB1eFgHiJkLm"
    }
  }
}
3
Toggle the server on
Cursor will show "Loading tools…" then list the Ratel tools (create_post, list_inbox, vote_poll, …). Open the chat sidebar and the agent can now use them.

Any client that speaks the Model Context Protocol over Streamable HTTP can connect. The endpoint accepts JSON-RPC 2.0 requests at POST {endpoint} and streams responses via Server-Sent Events.

1
Smoke-test with curl
Confirm your endpoint is reachable and lists tools.
terminal 
curl -X POST https://ratel.foundation/mcp/<your-token> \
  -H "Content-Type: application/json" \
  -H "Accept: application/json, text/event-stream" \
  -d '{"jsonrpc":"2.0","id":1,"method":"tools/list"}'
2
Use any MCP SDK
Python, TypeScript, Rust SDKs all work. Point the client's StreamableHTTPServerTransport at your endpoint URL.
