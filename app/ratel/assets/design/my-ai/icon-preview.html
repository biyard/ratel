1 · In-context · Arena topbar

The new My AI button sits between the like button and the panel toggles on the arena topbar. It uses a cyan/violet gradient outline so it reads as a distinct "external connection" affordance vs. the gold panel-toggles. A small green dot in the top-right corner signals that the user has an MCP secret generated and the endpoint is live.

2 · Button states

Three states: connected (token generated, dot is green), not connected (no secret yet, dot is dim), and active (currently on the My AI page — gold pressed style, matches existing panel toggles).

Connected
User has an active MCP secret. Click to copy or regenerate.
Not connected
No MCP secret generated yet. Page will prompt to generate one.
Active route
When the user is on /my-ai, button uses the standard pressed style.

3 · Behavior

My AI · Click target
On click the topbar navigates to /my-ai. The My AI page (next file) shows the user's MCP server URL, copy/regenerate actions, and a tabbed setup guide for Claude Code, Claude Desktop, Cursor, and a generic JSON-RPC config.
